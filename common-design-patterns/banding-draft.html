Full-width band: Canada.ca design system

  • Beta
  • Last updated: 2020-12-09

The full-width band is a grey background that extends from one edge of the screen to the other.

When to use

  • Use to visually group related content together
  • Use to visually divide a page into clear sections

What to avoid

Avoid using this pattern for purely decorative reasons.

Avoid changing the recommended banding application in templates that already include banding.

How to implement

[Coded examples of main variants, with optional specific When to to use and What to avoid for specific variants]

  1. Visible heading
  2. Hidden heading

1. Visible heading

When to use: Use when a heading is helpful to understand the grouped content

What to avoid: Do not include visible headings if the grouped content is clearly understood without them

2. Hidden header

When to use: Use when the grouped content is clearly understood as a section without a heading. Provide an invisible heading for screen readers who won't have a visual cue.

What to avoid: Do not make headings invisible if they contain key information to understanding to the content that follows.
