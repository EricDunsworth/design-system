Labels: Canada.ca design system

  • Last updated: 2020-12-02

Labels are visual indicators that assign a tag something (an item, a program, a service, etc.).

Primary label Success label Information label Warning label Danger label

On this page

  • When to use
  • What to avoid
  • How to implement
  • Latest changes
  • Discussion

When to use

Use labels to signal to that an item, program or service belongs to a category. For example, if some are open, some closed, and some will open soon, using labels to convey this information can be useful.

Make sure the design makes it clear the label applies to which item.

What to avoid

Don't use long labels or full sentences - keep them short.

Don't use labels that have overlapping meaning.

If you're trying to signal that a program is open or closed, there is no need to add the date that the status came in effect

How to implement

Use the right type of label:

  1. Primary
  2. Success
  3. Information
  4. Warning
  5. Danger

1. Primary label

When to use: Use to signal an important information about an item. For example, when a new program is not yet open but will soon be.

Canada Recovery Benefit (CRB) Opening October 12

Status: Opening October 12

Code 
<p>Some text here <span class="label label-primary">(Label info)</span></p>

2. Success label

When to use: use to signal an item is open or available

Protecting Canada’s Fresh Water

Open - Accepting input
Code 
<p>Some text here <span class="label label-success">(Label info)</span></p>

3. Information label

When to use: Use to tag an item with a value. For example, identifying a new blog post.

Labelling study: which words work best New

Code 
<p>Some text here <span class="label label-info">(Label info)</span></p>

4. Warning label

When to use: Use when you want to bring attention to an item to warn people of a change. For example, if a program is ending soon.

Canada Emergency Response Benefit (CERB Ending soon

Status: Closing soon: September 26

Code 
<p>Some text here <span class="label label-warning">(Label info)</span></p>

5. Danger label

When to use: Use when an item is no longer available. For example, when a program is no longer available.

Canada Emergency Response Benefit (CERB Closed

Status: Closed

Code 
<p>Some text here <span class="label label-danger">(Label info)</span></p>
